1. top

top은 리눅스 시스템의 현재 상태를 실시간으로 모니터링하는 명령어이다. CPU 사용률, 메모리 사용량, 실행 중인 프로세스 정보를 계속 갱신하며 보여준다.

주요 특징
- 실시간 CPU, 메모리 사용량 확인
- 프로세스 목록 확인
- 특정 프로세스의 우선순위 변경(nice)
- 프로세스 종료 기능 제공

사용 예시
 top


2. ps

ps는 현재 실행 중인 프로세스 정보를 출력하는 명령어이다. 특정 옵션과 함께 사용하면 시스템 전체 프로세스나 자세한 정보를 볼 수 있다.

자주 사용하는 옵션
- ps : 현재 터미널에서 실행 중인 프로세스 표시
- ps -e : 시스템 전체 프로세스 표시
- ps -f : 자세한 프로세스 정보(full format)
- ps aux : 사용자, CPU, 메모리 등 모든 정보 표시

사용 예시
 ps aux | grep python


3. jobs

jobs는 현재 쉘에서 실행 중인 백그라운드 작업을 확인하는 명령어이다. 쉘 내부에서 실행한 작업만 표시되며, 작업 번호를 통해 관리할 수 있다.

주요 특징
- 백그라운드 실행 작업 목록 확인
- 작업 번호와 상태 표시

사용 예시
 sleep 100 &
 jobs


4. kill

kill은 프로세스에 시그널을 보내 종료시키는 명령어이다. 기본 시그널은 SIGTERM(15)이며, 즉시 강제 종료 시에는 SIGKILL(9)을 사용한다.

대표적인 시그널
- 15 (SIGTERM): 정상 종료 요청
- 9 (SIGKILL): 강제 종료
- 2 (SIGINT): Ctrl + C와 동일한 인터럽트

사용 예시
정상 종료:
 kill 1234

강제 종료:
 kill -9 1234
